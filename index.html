Qu'est-ce que le Javascript ?

" Le JavaScript ou JS, inventé Brendan Eich et développé par Netscape, fait son apparition en 1995, sous le nom de LiveScript, dans le but de dynamiser les pages web. C'est un language de programmation dont l'utilisation s'est aujourd'hui largement répandue. "

alert( 'Hello, world!' );

un exemple de code en Javascript.


Particularités du language

" JavaScript est un langage de script léger, orienté objet, principalement connu comme le langage de script des pages web. Mais il est aussi utilisé dans de nombreux environnements extérieurs aux navigateurs web tels que Node.js. Le code JavaScript est interprété ou compilé à la volée (JIT). C'est un langage qui permet de programmer suivant plusieurs paradigmes de programmation : fonctionnelle, impérative et orientée objet.

var result = "";
var i = 0;
do {
i = i + 1;
result = result + i;
} while (i < 5);
console.log(result);

boucle "do .. while" basique en JavaScript


Avec ES6, Javascript à développé de vraie particularité dans la syntaxe du language ce qui permet d'écrire plus simplement ou de manière plus concise le code. Ce qui suit est un exemple de code écrit avec ES5 (l'ancienne norme Javascript, comparé à l'ES6 nouvelle norme :

function greetings (name) {
return 'hello ' + name
}

Voici un morceau de code en ES5.

const greetings = name => `hello ${name}`;

La même chose écrite en ES6.


Javascript & Java

" Il ne faut absolument pas confondre les deux! Le Java est un tout autre language. Pour citer quelques différences, tandis que le Javascript est un language non compilé, intégré directement dans le contenu HTML, dont l'extension des fichiers source est .js et nécéssitant peu de mémoire; Le Java quant à lui est un language orienté objet, compilé, qui utilise les classes et a besoin de plus de mémoires et dont l'extension des fichiers sources est .java "

int maVariable;
maVaariable = 5687;

déclaration d'une variable de type int en Java, le Javascript ne nécéssite pas de déclarer le type de la variable.


Node.js

" Node.js est une plateforme de développement Javascript. Ce n'est pas un serveur, ce n'est pas un framework, c'est juste le langage Javascript avec des bibliothèques permettant de réaliser des actions comme écrire sur la sortie standard, ouvrir/fermer des connections réseau ou encore créer un fichier. " En résumé Node.js sert à faire du développement back-end avec Javascript.

var http = require('http');
http.createServer(function (req, res) {
res.writeHead(200, {'Content-Type': 'text/plain'});
res.end('Hello World!');
}).listen(8080);

Exemple de code avec l'outil Node.js
